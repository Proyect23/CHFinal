Entrega Final


Este es un modelo de un site, que este pensado para la gestion de tareas dentro de un equipo, basado en un sistema de tarjetas para el almacenamiento de las actividades

Para el ingreso al site se debe crear un user o utilizar el admin.
En el mismo login encontraremos el form y un apartado legal, donde el usuario al crear la cuenta, acepta los terminos y condiciones de la plataforma.
En caso de no aceptarlas, este no podra registrarse.

Al crear la cuenta se nos pedira detailles como el username, email y un psw, luego este ya tendria acceso a un panel con distintas opciones CRUD para realizar.
Luego de obtener nuevas actualizaciones sobre las tareas asignadas, el user podra optar por marcarla como hecha o en su defecto eliminarla, en caso de que se encuentre algun error

La aplicacion de este site es la gestion, pero si se podria agregar opciones como: Inicio de sesiones con gmail, github, discord o similares
